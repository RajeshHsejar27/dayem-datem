𝟏.𝑬𝒙𝒕𝒓𝒂𝒄𝒕 𝒕𝒉𝒆 𝑪𝒐𝒏𝒕𝒆𝒏𝒕𝒔 𝒊𝒏𝒔𝒊𝒅𝒆 𝒂 𝒇𝒐𝒍𝒅𝒆𝒓. <br>
